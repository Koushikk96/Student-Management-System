# Student Management System

## Description
A console-based Student Management System developed in C++ using file handling.

## Features
- Add Student
- Display Students
- Search Student
- Update Student
- Delete Student
- Persistent Storage using Binary Files

## Technologies Used
- C++
- File Handling
- Structures
- Functions

## How to Run

Compile:

g++ main.cpp -o student

Run:

./student
